리스트에 순차적으로 접근 해야 할 경우 두 점의 위치를 기록하면서 처리하는 알고리즘
정렬 돼 있는 두 리스트의 합집합에도 사용함. merge sort의 기초가 되기도 함

예시)
    어떤 숫자들의 리스트가 주어질 때 해당 리스트의 연속 수열의 합이 특정 값을 가지는 것을 확인 할 경우
    1. 시작점과 끝점이 첫번째 원소를 가리키도록 한다.
    2. 현재 부분의 합이 M과 같다면 카운트
    3. 현재 부분의 합이 M보다 미만이면 끝부분을 1증가
    4. 현재 부분의 합이 M보다 이상이면 시작 부분을 1증가
    5. 반복
